有理函数积分是微积分中不定积分的重要类型，其核心思路是通过代数变形将复杂有理式拆解为简单可积形式，常用方法如下：

有理函数形如 P(x)Q(x)\frac{P(x)}{Q(x)}Q(x)P(x)​（P(x),Q(x)P(x),Q(x)P(x),Q(x) 为多项式，Q(x)≠0Q(x)\neq0Q(x)=0 ），核心是用部分分式分解，将其拆为：

P(x)Q(x)=多项式+∑Ai(x−ai)ki+∑Bjx+Cj(x2+pjx+qj)mj\frac{P(x)}{Q(x)} = \text{多项式} + \sum \frac{A_i}{(x - a_i)^{k_i}} + \sum \frac{B_jx + C_j}{(x^2 + p_jx + q_j)^{m_j}}Q(x)P(x)​=多项式+∑(x−ai​)ki​Ai​​+∑(x2+pj​x+qj​)mj​Bj​x+Cj​​

（其中 x2+pjx+qjx^2 + p_jx + q_jx2+pj​x+qj​ 是不可约二次式，pj2−4qj<0p_j^2 - 4q_j < 0pj2​−4qj​<0 ）

具体方法与步骤

  1. 多项式除法（假分式→多项式 + 真分式）

若 deg⁡P(x)≥deg⁡Q(x)\deg P(x) \geq \deg Q(x)degP(x)≥degQ(x)（分子次数**≥分母次数），先用多项式长除法**，将假分式拆为 “多项式 + 真分式”（真分式：分子次数 < 分母次数）。

例：∫x3+1x+1dx\int \frac{x^3 + 1}{x + 1}dx∫x+1x3+1​dx，先分解 x3+1x+1=x2−x+1\frac{x^3 + 1}{x + 1}=x^2 - x + 1x+1x3+1​=x2−x+1（多项式除法），再积分 ∫(x2−x+1)dx\int (x^2 - x + 1)dx∫(x2−x+1)dx。

  1. 部分分式分解（真分式→简单分式和）

对真分式 P(x)Q(x)\frac{P(x)}{Q(x)}Q(x)P(x)​，按分母 Q(x)Q(x)Q(x) 的因式分解形式分解：

    对应部分分式：Ax−a\frac{A}{x - a}x−aA​（AAA 为待定系数）。

  • **情况 2：分母含重一次因式 ** (x−a)k(x-a)^k(x−a)k

    对应部分分式：A1x−a+A2(x−a)2+⋯+Ak(x−a)k\frac{A_1}{x - a} + \frac{A_2}{(x - a)^2} + \dots + \frac{A_k}{(x - a)^k}x−aA1​​+(x−a)2A2​​+⋯+(x−a)kAk​​。

  • **情况 3：分母含不可约二次因式 ** x2+px+q,(p2−4q<0)x^2+px+q,(p^2-4q<0)x2+px+q,(p2−4q<0)
